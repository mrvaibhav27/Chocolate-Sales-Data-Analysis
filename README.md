# Chocolate Sales Data Analysis
## Project Overview
Sales data analysis using Excel to track revenue trends, product performance, and monthly sales insights through dashboards and pivot tables.

## Objectives
- Analyze overall sales 
- Identify top-performing chocolate products
- Track monthly and category-wise sales performance
- Create an easy-to-understand dashboard for management reporting

## Dataset
- Chocolate sales data
- Includes product name, category, quantity sold, sales amount
- Cleaned and structured for analysis in Excel

## Tools & Techniques Used
- Microsoft Excel  
- Pivot Tables  
- Pivot Charts  
- Data Cleaning & Formatting    
- Dashboard Design

## Analysis Performed
- trend analysis
- Product-wise and category-wise sales comparison
- Identification of top-selling and low-performing products
- Sales contribution analysis using sale person

## Dashboard
An interactive Excel dashboard was created to visualize:
- Total Sales 
- Sales Trends
- Top Products Performance
- Category-wise Sales Distribution

## Key Insights
- Identified peak sales and trends
- Provided insights to improve product focus and inventory planning
